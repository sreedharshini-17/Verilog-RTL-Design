# One-Bit ALU with Carry and Overflow

## Description
This project implements a one-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Operations
- AND
- OR
- XOR
- ADD

## Inputs
- A
- B
- Cin
- opcode

## Outputs
- result
- carry
- overflow

## Tools Used
- Xilinx ISE 14.7
- Spartan-6 FPGA

## FPGA Implementation
Implemented on Spartan-6 FPGA using UCF constraints file.
