# One-Bit ALU with Carry and Overflow

## Description
This project implements a one-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Operations
- AND
- OR
- XOR
- ADD

## Inputs
- A
- B
- Cin
- opcode

## Outputs
- result
- carry
- overflow

## Tools Used
- Xilinx ISE 14.7
- ISim Simulator
- Spartan-6 FPGA

## Learning Outcome
- RTL combinational design
- Opcode-based operation selection
- Carry and overflow handling
- Verilog simulation and debugging
